In Linux, everything is organized into a single-rooted hierarchical tree starting from / (root directory).

Important Directories:
Directory Description
/Root directory (top of the hierarchy)
/homeContains user home directories (e.g., /home/dirushan)
/binEssential system binaries (commands like ls, cat)
/sbinSystem binaries for admin (like reboot, shutdown)
/etcSystem-wide configuration files
/usrUser-installed software and libraries
/varVariable data (logs, mail, spool)
/tmpTemporary files (auto-deleted on reboot)
/devDevice files (like /dev/sda)
/procVirtual filesystem for process information
/lib, /lib64Essential shared libraries
/optOptional third-party software
/bootBootloader and kernel files
